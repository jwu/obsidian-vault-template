## 目录结构

```
~
├── bin/                # 存放一些配置相关的信息
├── src/                # 存放 github 等地方下载的源码
├── dev/                # 存放开发中的代码
└── work/               # 存放 obsidian 知识库
```

其中 `src/` 目录下，我们按照 `项目名@作者` 的方式下载 github 等地方的源码并管理

```
src/
├── bevy@bevyengine/
├── godot@godotengine/
└── zed@zed-industries/
```
